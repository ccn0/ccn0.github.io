Engines

View long-winded tutorial here: You can find the search results page by searching something, then looking for it in the address bar. Enter a URL for a search engine, e.g. https://www.google.com/search?q=hello, and replace the query with %s, e.g. https://www.google.com/search?q=%s. This URL will replace the %s or however many you add, with your search query. Enter a symbol, e.g. @g so that you can place the symbol before the query to specify the search engine.

By the way, you don't have to use a %s.

Basically, the whole idea is that instead of relying on your browsers search "utilities," you can go ahead and use SearchPlusPREMIUM which despite the name sounding like otherwise, is actually free. You can add it by entering the URL (if %s is the query), https://ccn0.net/things/searchpluspremium/search.html?q=%s, or if your using Firefox, click on the search bar while on this page and click the little icon with the green plus.
