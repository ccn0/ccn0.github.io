Roomer



play on turbowarp ?

Instructions

Make a room with furniture. Click [Edit] Button to start.

Controls:
- Drag 'Furniture' around with Mouse
- Q to size up
- E to size down
- W to set to 100%
- A + D to switch item
- S to place item
- 1 to rotate left
- 3 to rotate right
- 2 to set reset direction
- Z to set brightness up
- C to set brightness down
- X to reset brightness
- Arrow Keys to move 'Furniture' precisely
