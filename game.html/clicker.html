Instructions

click the black dot

press space to debug
press q + w to see running code
when hotspot equals 1000 game ends

z turn on flashlight
x turns off flashlight
s turns on sweeper
a turns of sweeper

codes:
find dot - shows where the dot is
send dot - puts the dot at the center
auto - spawns the clicker
find com - shows where com is
drag all - sets everything to draggable
unauto - disable clicker
unfind dot - disables easy finding the dot
unfind com - disables find com
erase - erases dot's web
vs clicker - battle the clicker by grabbing as much points as you can
color #adj #color - change dots path color
size dot #plmi #num - change dot size
compass #onoff - points to the dot
explore - blacks out stage

there are secret codes aswell
