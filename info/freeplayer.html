FREEPLAYER

status: in-development | play beta

FREEPLAYER is a tool to design games and play them in the same game shared using a code. This game will not be available for a little while due to coding and designing test runs of it.


Controls

Create Menu
[W,A,S,D] to move drawing cursor
[X] to turn on draw
[Z] to turn off draw
[SPACE] to click
[ARROW KEYS] to move clicking cursor.
