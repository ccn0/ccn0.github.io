MenuEngine Docs

How to use MenuEngine to it's fullest.

Necessities: You import your own assets, follow the same principals as the templates provided. You must add a semicolon ; at the end of every line. Seperate values using a semicolon and space. e.g. #template; center;. You should add a @~closemenu; at the top of your script if your menu is opened via a different method.
Headings: You can create headings by adding "#" at the beginning of a line. e.g. #template;. You can adjust the position using left right or center. e.g. #template: center;.
Buttons: You can create buttons by adding "@" at the beginning of a line. e.g. @template;. You must add a second value for whatever event you want to broadcast. e.g. @template; event; The third value is the position of the button, such as left right or center. e.g. @template; event; center; would create a centered button that broadcasts "event" when clicked.
Boolean: You can create switches by adding ">" at the beginning of a line. e.g. >template;. This first value is the slot in the ~FOR list. It can only be 1 of 2 values, 0 or 1 (binary). The next value is the position, such as left right or center. e.g. >template; center; would create a switch centered within the stage.
Margin Radius: Add a margin radius using "!" at the beginning of a line. This bounds elements to a certain space. e.g. !; 100; would space every object within 100 pixels from the center of the stage.
