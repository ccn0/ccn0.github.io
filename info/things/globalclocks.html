    About

  • GlobalClocks is a countdown/count-up (?) maker where you enter a date and you can share a link to see how long until or after a date is.

    • Usage

  • Make sure the Create GUI is open by checking in the URL if it contains #create. You can add a title or a description. Enter a date and time. Check if you want Embed code or a link to the countdown/count-up. Then share or use this code wherever you want it to be.
