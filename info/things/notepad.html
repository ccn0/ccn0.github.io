    • About

  • The Notepad family of Things are text editors designed for specific purposes. They are loosely related by the shared function of text manipulation, but some have other features.

    • CCN0TEPAD

  • CCN0TEPAD, originally Notepad, is a text editor designed for a younger me, back when i didnt know anything. It has features like making the notepad taller, erasing everything, copying all, and a shitty dark theme.

    • Notepad Premium

  • Notepad Premium, a.k.a Notepad Light, is a text editor designed for being used locally or offline. It has no JavaScript features unlike every other Notepad. It only has a light mode, and has rarely been updated since its inception.

    • Notepad Win

  • Notepad Win was designed to be similar in functionality to the Windows 10 notepad, but never got there. It instead had many features seen in Programmable Notepad, keyboard shortcuts, dark mode, text wrapping, and saving/loading files.

    • Programmable Notepad

  • Programmable Notepad is a text editor with commands. You can replace text, change the font and styles, and run JavaScript for automating tasks or inserting math/arithmetic.
