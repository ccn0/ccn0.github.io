About:

Instead of relying on your browsers search "keywords," you can instead use Google 2, which despite the name sounding like otherwise, is actually free. You can add it as a search engine by using the URL, https://ccn0.net/things/searchpluspremium/search.html?q=%s, or if your using Firefox, click on the search bar while on this page and click the little icon with the green plus.

Read tutorial:
  • Type the name of a search engine into the Quick Add bar and hit enter.
    • OR
  • Click Add New
  • Add a URL, with %s in the place of the query, e.g. https://www.google.com/search?q=%s
  • Add a Symbol, which you use to specify engine, e.g. searching @ddg hello world will use DuckDuckGo
  • The default search engine will be the one at the top. %s can be used any amount of times.
